## Birthdays - CC50 - Utilizando Flask e BD

1 - Abra o git-bash na pasta do projeto desejado;

2 - use o comando:

 	wget https://cdn.cs50.net/2020/fall/labs/9/lab9.zip

3 - use o comando:
 
	unzip lab9.zip

4 - use o comando:

	rm lab9.zip
  
5 - use o comando:

	ls
  
6 - use o comando:

	pip install Flask
  
7 - use o comando:

	pip3 install cs50
  
7 - use o comando:

	cd lab9
  
10 - use o comando:

	ls
